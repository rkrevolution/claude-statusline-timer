# LinkedIn post (short / punchy)

I kept losing track of how much time I was spending in Claude Code. Hit the 5-hour rate limit a few too many times. Built a status bar for myself.

It sits in the terminal and shows real daily + weekly time across every session, correctly handling parallel workflows (git worktrees, subagents). No double-counting, no idle time inflation, no API tokens consumed.

If you live in Claude Code, grab it: https://github.com/rkrevolution/claude-statusline-timer
